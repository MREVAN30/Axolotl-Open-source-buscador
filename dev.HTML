Axolotl.dev

O Axolotl é um sistema de motor de busca Open Source desenvolvido para proporcionar pesquisas rápidas e privadas, priorizando a segurança e eficiência dos usuários.

Documentação do Ambiente

Rede de Busca Alternativa (R.B.A)

A R.B.A é um recurso avançado do nosso sistema que permite realizar pesquisas em bancos de dados JSON de forma integrada e eficiente através do Axolotl. 

{
    "name": "BambooHR",
    "url": "https://www.bamboohr.com",
    "description": "Software de gerenciamento de recursos humanos."
}

Esta é a estrutura padrão de uma R.B.A, otimizada para máxima eficiência.

API Axolotl

Disponibilizamos uma API gratuita para integração com nosso banco de dados de pesquisa.
Acesse através do endpoint:

./data/main.json
